FirebaseRepository

class FirebaseRepository

A Repository class used to mediate between Firebase and the rest of the application

This class can provide some Firebase references commonly used by the application such as :

  • The current FirebaseUser signed in to the application
  • A DocumentReference
  • A CollectionReference
  • A StorageReference
