firebaseSignInChildWithGoogle

fun firebaseSignInChildWithGoogle(googleAuthCredential: AuthCredential?, parentUID: String): MutableLiveData<UserData>?

Signs in a child user to the application with Google authentication credentials

This method launches the signInWithCredential task. If the task is successful, it signs in the user to the application and creates the UserData instance to return.

Parameters

googleAuthCredential - The AuthCredential used to authenticate the user

parentUID - The uid of the parent FirebaseUser

Return
the UserData instance of the child user who just signed in, held in a MutableLiveData class
