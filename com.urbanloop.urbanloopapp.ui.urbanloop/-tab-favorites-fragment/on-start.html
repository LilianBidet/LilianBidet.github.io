onStart

fun onStart(): Unit

onStart method overriden to interact with the Firebase database and get realtime updates of the child's data

This method calls addSnapshotListener from the DocumentReference referencing the child's data to listen to its changes. When the DocumentSnapshot is updated, it creates a UserData instance created from the DocumentSnapshot and updates the TabFavoritesViewModel data from it with the following methods :

  • TabFavoritesViewModel.updateIsCurrentUserChild
  • TabFavoritesViewModel.updateFavoriteDestinationsValue
  • TabFavoritesViewModel.updateCurrentCapsuleTypeWithUserData
  • TabFavoritesViewModel.updateAllowedDestinationsWithUserData

It also :

  • Updates the UI to make the default capsule type selected on the screen.
  • calls setUpRecyclerView when view model has been updated.
  • begins to listen to data with the adapter instance.
