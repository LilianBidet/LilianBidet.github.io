goToStation

private fun goToStation(documentSnapshot: DocumentSnapshot): Unit

Get a ticket for the selected station and navigates to the TicketFragment when it's done.

This method does the following :

  • Update the Firebase database with the new destination and capsule type with TabFavoritesViewModel.updateFirebaseCurrentDestination and TabFavoritesViewModel.updateFirebaseCurrentCapsuleType.
  • Add a new document corresponding to the new ticket in the Firebase database with the add method.
  • Add a new file in the Firebase storage that stores the QR code with the putBytes(https://firebase.google.com/docs/reference/kotlin/com/google/firebase/storage/StorageReference#putbytes) method.
  • Navigate to the TicketFragment with the navigate

Parameters

documentSnapshot - The DocumentSnapshot which corresponds to the station item
