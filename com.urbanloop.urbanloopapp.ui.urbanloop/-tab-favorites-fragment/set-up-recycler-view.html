setUpRecyclerView

private fun setUpRecyclerView(): Unit

Set up the RecyclerView

This method set up the RecyclerView by doing the following :

  • Configure the final query
  • Build the FirestoreRecyclerOptions with the final query
  • Instantiate a FirestoreRecyclerAdapter with these options to initialize adapter
  • Optimize the RecyclerView with setHasFixedSize
  • Set up the LayoutManager of the RecyclerView
  • Set up the Adapter of the RecyclerView with the adapter instance
