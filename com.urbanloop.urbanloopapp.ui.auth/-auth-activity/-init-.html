AuthActivity()

An AppCompatActivity class providing an authentication interface to the user

This activity provides an authentication interface to the user made up of the following elements :

  • A section to sign in with an e-mail and a password
  • A section to sign in with other providers like Google

To match the recommended app architecture, this class interacts with the AuthViewModel to handle it's data logic.
