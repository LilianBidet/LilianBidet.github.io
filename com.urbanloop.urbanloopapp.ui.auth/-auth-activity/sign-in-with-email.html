signInWithEmail

private fun signInWithEmail(): Unit

Signs in a user from the email and password entered in the appropriate fields

This method gets the text contained in the email and password fields and starts AuthViewModel.signInWithEmail to sign in the user from these credentials. Then, two cases are handled :

  • If the authenticated user is a new user, it starts createNewUser to create the user in the Firebase database
  • Else, authentication is done so it starts goToMainActivity
