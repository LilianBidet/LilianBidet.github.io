drawItinerary

private fun drawItinerary(origin: LatLng, dest: LatLng): Unit

Draws an itinerary between two coordinates

First, it asks Google for the JSON file containing the points you're going to pass through using FetchURL Then, it parses the JSON file to get the points using ParserTask using DataParser Finally, ParserTask creates a polyline that will be displayed on the map

Parameters

origin - The origin position of the itinerary

dest - The destination position of the itinerary
