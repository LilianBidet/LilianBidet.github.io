firebaseSignInWithEmail

fun firebaseSignInWithEmail(email: String, password: String): MutableLiveData<UserData>?

Signs in a user to the application with E-mail/password credentials

This method launches the createUserWithEmailAndPassword task. Then, two cases are handled :

  • If the task is successful, it creates a new user account with the given email address and password and a UserData instance to return.
  • If the task is not successful, it catches the task exception. If this exception is a FirebaseAuthUserCollisionException, it launches the signInWithEmailAndPassword task to sign in the user with the given email address and password and create the UserData instance to return using user data retrieved from the Firebase database.

Parameters

email - The email address used to authenticate the user

password - The password used to authenticate the user

Return
the UserData instance of the user who just signed in, held in a MutableLiveData class
