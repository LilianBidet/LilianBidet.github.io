createUserInFirebaseDatabaseIfNotExists

fun createUserInFirebaseDatabaseIfNotExists(authenticatedUser: UserData): MutableLiveData<UserData>?

Stores user data from the given UserData instance in the Firebase database.

This method gets the DocumentReference of the given user by launching a get task. If no documents exists for this DocumentReference, it creates a document by launching a set task.

Parameters

authenticatedUser - the UserData instance used to create the document

Return
the UserData instance of the user for whom a document has just be stored in the Firebase database
