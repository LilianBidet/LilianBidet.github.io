SplashRepository

class SplashRepository

A Repository class used to mediate between Firebase and the rest of the application to know the authentication state of the application in the SplashActivity

This class can :

  • Check if a FirebaseUser is authenticated
  • Updates the UserData instance used by the SplashViewModel
