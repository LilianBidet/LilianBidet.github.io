createChildUserInFirebaseDatabaseIfNotExists

fun createChildUserInFirebaseDatabaseIfNotExists(authenticatedUser: UserData, nameInput: String): MutableLiveData<UserData>?

Stores user data from the given UserData instance in the Firebase database.

This method gets the DocumentReference of the given user by launching a get task. If no documents exists for this DocumentReference, it creates a document by launching a set task. Then, it updates the name field if necessary with an update task.

Parameters

authenticatedUser - the child UserData instance used to create the document

nameInput - the name used for the creation of the child account

Return
the UserData instance of the child user for whom a document has just be stored in the Firebase database
