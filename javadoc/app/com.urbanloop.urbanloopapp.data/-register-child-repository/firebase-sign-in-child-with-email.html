firebaseSignInChildWithEmail

fun firebaseSignInChildWithEmail(email: String, password: String, parentUID: String): MutableLiveData<UserData>?

Signs in a child user to the application with E-mail/password credentials

This method launches the createUserWithEmailAndPassword task. If the task is successful, it creates a new user account with the given email address and password, signs in the user to the application and creates the UserData instance to return.

Parameters

email - the email address used to create the child account

password - the password used to create the child account

parentUID - the uid of the parent FirebaseUser

Return
the UserData instance of the child user who just signed in, held in a MutableLiveData class
