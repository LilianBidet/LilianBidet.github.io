RegisterChildRepository

class RegisterChildRepository

A Repository class used to mediate between Firebase and the rest of the application to authenticate child users in the RegisterChildActivity

This class can perform the Firebase authentication process for the creation of children accounts with the methods below :

  • Google authentication
  • E-mail/password authentication

It also stores user data into the Firebase database linked to the application
