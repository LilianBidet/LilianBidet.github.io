MapFragment

class MapFragment : Fragment, OnMapReadyCallback, OnMarkerClickListener

A Fragment class providing a user interface to reach Urbanloop stations from a map view.

This fragment provides a user interface to reach Urbanloop stations from a map view. The interface is made up of the following elements :

  • A map view displayed with the Google Maps API
  • A button allowing the user to display an itinerary to the nearest station on the map view

To match the recommended app architecture, this class interacts with the MapViewModel to handle it's data logic.
