MainActivity

class MainActivity : AppCompatActivity, AuthStateListener

An AppCompatActivity class providing the main features to use Urbanloop.

This activity provides the main features to use Urbanloop by navigating through different fragments with a Navigation Drawer. The home fragment is the UrbanloopFragment This navigation drawer redirects to the following fragments.

  • UrbanloopFragment
  • MapFragment
  • RideFragment
  • TicketFragment
  • ProfileFragment
  • ChildrenAccountsFragment
