updateTicketInformation

fun updateTicketInformation(userData: UserData): Unit

Updates the currentDestination and the currentCapsuleType values with a UserData instance.

This method calls the FirebaseRepository.getStationDataReferenceFromID with the id of the current destination stored in the UserData instance in order to get its information from the Firebase database with the get method. Then, it can update the currentDestination value from the StationData instance retrieved.

Parameters

userData - The UserData instance used to update the values.
