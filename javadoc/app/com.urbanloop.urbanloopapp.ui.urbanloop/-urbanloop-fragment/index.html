UrbanloopFragment

class UrbanloopFragment : Fragment

A Fragment class providing a user interface to navigate through the urbanloop stations and interact with them to take a ride among three tabs:

  • Stations : all the urbanloop stations are referenced in this tab.
  • Favoris : Only the stations marked as favorite by the user are referenced in this tab.
  • Recents : Only the recent destinations are referenced in this tab.

This fragment provides a user interface to see the favorite urbanloop stations of the user and interact with them to take a ride. The interface is made up of the following elements :

  • A section to show the urbanloop stations among the three tabs described above.

See these fragments for more information about the content of the tabs :

  • TabStationsFragment
  • TabFavoritesFragment
  • TabRecentFragment

To match the recommended app architecture, this class interacts with the UrbanloopViewModel to handle it's data logic.
