TabRecentFragment

class TabRecentFragment : Fragment

A Fragment class providing a user interface to see the urbanloop stations recently used by the user and interact with them to take a ride.

This fragment provides a user interface to see the urbanloop stations recently used by the user and interact with them to take a ride. The interface is made up of the following elements :

  • A section to show the urbanloop stations
  • A section to see and select the capsule type to use for the ride

This fragment corresponds to the "Recents" tab in the UrbanloopFragment ViewPager.
