TabStationsFragment

class TabStationsFragment : Fragment

A Fragment class providing a user interface to see the urbanloop stations and interact with them to take a ride.

This fragment provides a user interface to see the urbanloop stations and interact with them to take a ride. The interface is made up of the following elements :

  • A section to show the urbanloop stations
  • A section to see and select the capsule type to use for the ride

This fragment corresponds to the "Stations" tab in the UrbanloopFragment ViewPager.

To match the recommended app architecture, this class interacts with the TabStationsViewModel to handle it's data logic.
