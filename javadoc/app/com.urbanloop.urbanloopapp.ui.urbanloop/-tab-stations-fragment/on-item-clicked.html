onItemClicked

private fun onItemClicked(documentSnapshot: DocumentSnapshot): Unit

Handles the click on a station "Go" button.

This method is called when the "Go" button is clicked inside of a station item. First, it checks if the user is a child.

If the user is a child :

It checks if the the station clicked belongs to the allowed stations for this user and set a boolean accordingly.

  • If the station is allowed, it calls goToStation.
  • If the station is not allowed, it displays a Snackbar to warn the user.

If the user is not a child :

It calls goToStation.

Parameters

documentSnapshot - The DocumentSnapshot which corresponds to the station item
