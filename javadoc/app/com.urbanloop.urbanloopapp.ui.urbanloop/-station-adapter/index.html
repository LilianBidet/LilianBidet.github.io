StationAdapter

class StationAdapter : FirestoreRecyclerAdapter<StationData, StationViewHolder>

A FirestoreRecyclerAdapter class to populate the RecyclerViews in charge of showing the stations items in the TabStationsFragment and TabFavoritesFragment

This class handles the population of the RecyclerViews in charge of showing the stations items in the TabStationsFragment and TabFavoritesFragment. To be able to do so, it contains a ViewHolder class and methods associated to it that populate the final view.
