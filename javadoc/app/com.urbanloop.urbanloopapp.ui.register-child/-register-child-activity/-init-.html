RegisterChildActivity()

An AppCompatActivity class providing an authentication interface to authenticate a child user from a parent account.

This activity provides an authentication interface to authenticate a child user made up of the following elements :

  • A section to describe the process to the current user
  • A section to sign in with an e-mail and a password
  • A section to sign in with other providers like Google

To match the recommended app architecture, this class interacts with the RegisterChildViewModel to handle it's data logic.
