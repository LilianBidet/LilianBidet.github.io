onLoadButtonClicked

private fun onLoadButtonClicked(): Unit

Listen to the Load button

This method calls setOnClickListener from the button view to invoke a callback when the button is clicked. The callback calls the ChildManagementViewModel.increaseBalance method to increase the child's balance when the button is clicked.
