ChildrenAccountsFragment

class ChildrenAccountsFragment : Fragment

A Fragment class providing a user interface to manage children accounts.

This fragment provides a user interface to manage children accounts made up of the following elements :

  • A header section where you can click on a button to create a child account
  • A section that shows all the children accounts linked to the current user and allows to manage each one of them by clicking on a button

To match the recommended app architecture, this class interacts with the ChildrenAccountsViewModel to handle it's data logic.
