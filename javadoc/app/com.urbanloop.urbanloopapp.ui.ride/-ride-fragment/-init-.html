RideFragment()

A Fragment class providing a user interface to see the current ride information.

This fragment provides a user interface to see the current ride information made up of the following elements :

  • A section to display the destination and the type of capsule used for the ride
  • A view of the current position on the ride, the previous station and the next one.
  • A section displaying information like the last station or the expected arrival time
  • Two buttons, one to change the current destination and the other to make an emergency call
