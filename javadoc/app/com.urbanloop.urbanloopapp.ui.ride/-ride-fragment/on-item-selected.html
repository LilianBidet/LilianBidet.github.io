onItemSelected

fun onItemSelected(parent: AdapterView<*>?, view: View?, position: Int, id: Long): Unit

onItemSelected method overridden to handle the Spinner selection in the AlertDialog.

This method gets the DocumentReference referencing the station associated with the clicked item in the Spinner and calls RideViewModel.updateViewModelNewDestination to update the current destination accordingly.

Parameters

parent - The AdapterView where the selection happened

view - The view within the AdapterView that was clicked

position - The position of the view in the adapter

id - The row id of the item that is selected
