private fun onPersonalInfoCardView(): Unit

Listen to the button that redirects to the PersonalInformationFragment.

This method calls setOnClickListener from the button view to invoke a callback when the button is clicked. Then, the callback navigates to the PersonalInformationFragment with the navigate method.
