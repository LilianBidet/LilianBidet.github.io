getGoogleAuthCredentialAndDeleteAccount

private fun getGoogleAuthCredentialAndDeleteAccount(googleSignInAccount: GoogleSignInAccount): Unit

Gets the AuthCredential from a provided GoogleSignInAccount to call reAuthGoogleAndDelete with it.

This method gets the AuthCredential from the provided GoogleSignInAccount using the getCredential method from the GoogleAuthProvider class. Then, it starts reAuthGoogleAndDelete to re-authenticate the user and delete the account from the Firebase Authentication system.

Parameters

googleSignInAccount - The GoogleSignInAccount used to get the AuthCredential.
