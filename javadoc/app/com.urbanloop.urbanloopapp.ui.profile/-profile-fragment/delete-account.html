deleteAccount

private fun deleteAccount(): Unit

Deletes the account from the application.

This method delete the account from the application by removing the user from the Firebase system. Two cases are handled :

If the user has a Google account :

  • First it removes user data from the Firebase Database with the DocumentReference.delete method.
  • Then, it removes user data from the Firebase Storage with the StorageReference.delete method.
  • Finally, it calls reAuthAndDeleteWithGoogle to delete the user from the Firebase Authentication system.

If the user has an "email/password" account :

  • It calls reAuthAndDeleteWithEmail to delete the user from the Firebase Authentication system.
