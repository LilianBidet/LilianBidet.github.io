onCreateView

fun onCreateView(inflater: LayoutInflater, container: ViewGroup?, savedInstanceState: Bundle?): View?

onCreateView overridden to instantiate the user interface view

This method initializes the fragment by doing the following :

  • Set the action bar Title
  • Inflate the view using Data Binding
  • Enable the OptionsMenu with the setHasOptionsMenu method
  • Initialize the ProfileViewModel instance with initViewModel
  • Set up the visibility of the button allowing the user to manage the balance according to the child nature of the user
  • Set up the UI listeners with : onLoadBalance,onDefaultCapsuleDropDownButton,onPersonalInfoCardView,onPaymentCardView and onDeleteAccountButton.
