onStart

fun onStart(): Unit

onStart method overridden to interact with the Firebase database and get realtime updates of the user data.

This method calls addSnapshotListener from the DocumentReference referencing the user data to listen to its changes. When the DocumentSnapshot is updated, it creates a UserData instance created from the DocumentSnapshot and updates the ProfileViewModel data from it with the following methods :

  • ProfileViewModel.updateBalance
  • ProfileViewModel.updateEmail
  • ProfileViewModel.updateName
  • ProfileViewModel.updateDefaultCapsuleType It also sets the appropriate default capsule according to the UserData instance.
