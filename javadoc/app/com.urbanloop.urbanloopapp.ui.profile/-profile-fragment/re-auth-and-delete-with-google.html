reAuthAndDeleteWithGoogle

private fun reAuthAndDeleteWithGoogle(): Unit

Re-authenticates a Google user in order to delete the account from the Firebase Authentication system.

This method creates a GoogleSignInOptions instance to configure the Google Sign In API with its nested Builder class. Then, it calls the getClient method from the GoogleSignIn class to initialize the GoogleSignInClient instance.

When it's done, it gets an Intent to start the Google sign in flow from the GoogleSignInClient.getSignInIntent method and starts it by calling the startActivityForResult method.
