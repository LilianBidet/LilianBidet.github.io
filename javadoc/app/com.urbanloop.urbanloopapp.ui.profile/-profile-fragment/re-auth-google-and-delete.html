reAuthGoogleAndDelete

private fun reAuthGoogleAndDelete(credential: AuthCredential): Unit

Re-authenticates a user with an AuthCredential and delete the associated account.

This method calls the reauthenticate method to re-authenticate the user with the provided AuthCredential. Then, it deletes the account from the Firebase Authentication system with the delete method.

Parameters

credential - The AuthCredential used to re-authenticate the user.
