increaseBalance

fun increaseBalance(user: FirebaseUser): Unit

Updates the balance for the provided FirebaseUser in the Firebase database

This method calls FirebaseRepository.getUserDataReferenceFromUser with the provided FirebaseUser and updates the corresponding document in the Firebase database with the update method.

The balance is increased by 10 here.

Parameters

user - The FirebaseUser of whom it updates data.
