TicketFragment()

A Fragment class providing a user interface to see the current ticket and share it

This fragment provides a user interface see the current ticket and share it made up of the following elements :

  • A section to show the ticket information (destination, capsule...)
  • A section to show the QR Code of the ticket
  • A section to inform the user if NFC can be used
  • A button to share the ticket

To match the recommended app architecture, this class interacts with the TicketViewModel to handle it's data logic.
