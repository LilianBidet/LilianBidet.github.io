onStart

fun onStart(): Unit

onStart method overridden to get a DocumentReference referencing the user's data from the Firebase database.

This method gets a DocumentReference referencing the user's data from the Firebase database and updates the ticket TicketViewModel data according to it with the TicketViewModel.updateTicketInformation method. It also adapt the current capsule type icon displayed on the screen according to the UserData instance retrieved from the Firebase database.
