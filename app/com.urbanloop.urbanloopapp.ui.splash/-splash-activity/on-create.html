onCreate

protected fun onCreate(savedInstanceState: Bundle?): Unit

onCreate method overridden to perform the startup logic of the activity

This method initializes the activity by doing the following :

  • Inflate the activity's UI with setContentView
  • Initialize the FirebaseAuth instance
  • Initialize the SplashViewModel instance with the initSplashViewModel method
  • Check if a user is authenticated with checkIfUserIsAuthenticated
