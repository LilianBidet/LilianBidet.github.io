onStart

fun onStart(): Unit

onStart method overriden to interact with the Firebase database and get realtime updates of the child's data

This method calls addSnapshotListener from the DocumentReference referencing the child's data to listen to its changes. When the DocumentSnapshot is updated, it creates a UserData instance created from the DocumentSnapshot and updates the ChildManagementViewModel data from it with the following methods :

  • ChildManagementViewModel.updateBalance
  • ChildManagementViewModel.updateChildID
  • ChildManagementViewModel.updateChildName
  • ChildManagementViewModel.updateAllowedDestinations

It also updates the UI to make the stations checkboxes match the allowed destinations retrieved from the UserData instance
