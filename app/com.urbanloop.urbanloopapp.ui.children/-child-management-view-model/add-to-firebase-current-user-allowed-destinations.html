addToFirebaseCurrentUserAllowedDestinations

private fun addToFirebaseCurrentUserAllowedDestinations(userID: String, destinationName: String): Unit

Adds a destination to the allowed destinations of a user who corresponds to the provided user ID in the Firebase database.

This method calls FirebaseRepository.getStationsDataReference to get the CollectionReference corresponding to the stations and filter it to get the station which corresponds to the provided destination name with whereEqualTo.

Then, the station id is added to the allowedDestination field of the user data which corresponds to the provided user ID with the following methods :

  • update
  • arrayUnion

Parameters

userID - The uid of the FirebaseUser of whom it updates data

destinationName - The name of the added destination
