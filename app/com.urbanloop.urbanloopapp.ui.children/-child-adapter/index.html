ChildAdapter

class ChildAdapter : FirestoreRecyclerAdapter<UserData, ChildViewHolder>

A FirestoreRecyclerAdapter class to populate the RecyclerView in charge of showing the children items in the ChildrenAccountsFragment

This class handles the population of the RecyclerView in charge of showing the children items in the ChildrenAccountsFragment. To be able to do so, it contains a ViewHolder class and methods associated to it that populate the final view.
