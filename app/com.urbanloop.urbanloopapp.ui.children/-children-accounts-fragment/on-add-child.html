onAddChild

private fun onAddChild(): Unit

Listen to the Add button

This method calls setOnClickListener from the button view to invoke a callback when the button is clicked. Then, two cases are handled :

  • If the user is not a child, the callback calls goToRegisterChildActivity to go to the RegisterChildActivity
  • If the user is a child, it shows a Snackbar to warn the child about the impossibility to use this functionality.
