onCreateView

fun onCreateView(inflater: LayoutInflater, container: ViewGroup?, savedInstanceState: Bundle?): View?

onCreateView overridden to instantiate the user interface view

This method initializes the fragment by doing the following :

  • Set the action bar Title
  • Enable the OptionsMenu with the setHasOptionsMenu method
  • Inflate the view using Data Binding
  • Initialize the TicketViewModel instance with initViewModel
  • Checks the nfc activation with nfcCheck
  • Initialize the FirebaseAuth instance
  • Retrieve the QR Code of the ticket with the getFile method and shows it on the screen with the setImageBitmap method.
  • Save the image Uri of the QR Code with saveImage
  • Set up the UI listeners with : onTicketShareButton
