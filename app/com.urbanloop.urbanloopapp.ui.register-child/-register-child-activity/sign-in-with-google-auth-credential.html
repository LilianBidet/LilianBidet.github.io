signInWithGoogleAuthCredential

private fun signInWithGoogleAuthCredential(googleAuthCredential: AuthCredential): Unit

Signs in a user from a Google AuthCredential

This method starts RegisterChildViewModel.signInWithGoogle to sign in the user from the Google AuthCredential Then, two cases are handled :

  • If the authenticated user is a new user, it starts createNewUser to create the user in the Firebase database
  • Else, authentication is done so it starts goToMainActivity

Parameters

googleAuthCredential - The AuthCredential used to sign in.
