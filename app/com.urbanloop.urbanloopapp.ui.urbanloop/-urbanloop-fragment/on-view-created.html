onViewCreated

fun onViewCreated(view: View, savedInstanceState: Bundle?): Unit

onViewCreated method overridden to set up the ViewPager.

To do so, this method does the following :

  • Instantiate the StationsPagerAdapter
  • Set up the ViewPager and its adapter.
  • Listen to the tab changes with the addOnPageChangeListener
  • Get the favorite destinations of the user from the Firebase database with the get method to know if the view pager must initialize on the favorite tab.

See Create swipe views with tabs using ViewPager for more information
