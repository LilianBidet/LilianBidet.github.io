onDefaultCapsuleDropDownButton

private fun onDefaultCapsuleDropDownButton(): Unit

Listen to the dropdown button that shows the interface allowing to select a default capsule.

This method calls setOnClickListener from the button view to invoke a callback when the button is clicked. Then, two cases are handled :

  • If the view of the interface is not expanded, it calls ProfileViewModel.onDefaultCapsuleDropDownButton, set the visibility of the interface to View.VISIBLE and change the dropdown icon displayed.
  • If the view of the interface is expanded, it calls ProfileViewModel.onDefaultCapsuleDropDownButton, set the visibility of the interface to View.GONE and change the dropdown icon displayed.
