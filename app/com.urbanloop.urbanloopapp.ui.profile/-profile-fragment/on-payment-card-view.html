private fun onPaymentCardView(): Unit

Listen to the button that redirects to the payment ways fragment.

This method calls setOnClickListener from the button view to invoke a callback when the button is clicked. Then, the callback shows a Snackbar to tell the user that the functionality is not implemented yet. (for now)
