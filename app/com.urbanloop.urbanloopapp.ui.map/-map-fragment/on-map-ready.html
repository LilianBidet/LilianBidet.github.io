onMapReady

fun onMapReady(map: GoogleMap?): Unit

onMapReady method overridden to configure the map

This method configures the map dipslayed on the screen by doing the following :

  • Set a callback that's invoked when a marker is clicked with setOnMarkerClickListener
  • Enable the my-location layer with setMyLocationEnabled
  • Enable the my-location button with setMyLocationButtonEnabled
  • Show Urbanloop stations on the map with showStations
  • Get the device location and place the camera on it with getDeviceLocation
  • Listen to the last location of the user with the LocationManager.requestLocationUpdates method

Parameters

map - A non-null instance of a GoogleMap associated with the MapFragment or MapView that defines the callback.
