AuthRepository

class AuthRepository

A Repository class used to mediate between Firebase and the rest of the application to authenticate users in the AuthActivity

This class can perform the Firebase authentication process for the sign-in methods below :

  • Google authentication
  • E-mail/password authentication

It also stores user data into the Firebase database linked to the application
