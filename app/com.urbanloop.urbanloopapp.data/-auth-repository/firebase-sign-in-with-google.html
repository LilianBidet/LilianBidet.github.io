firebaseSignInWithGoogle

fun firebaseSignInWithGoogle(googleAuthCredential: AuthCredential?): MutableLiveData<UserData>?

Signs in a user to the application with Google authentication credentials

This method launches the signInWithCredential task. If the task is successful, it signs in the user to the application and creates the UserData instance to return. If the user is already registered in the Firebase Authentication system of the application, it creates the UserData instance using user data retrieved from the Firebase database.

Parameters

googleAuthCredential - The AuthCredential used to authenticate the user

Return
The UserData instance of the user who just signed in, held in a MutableLiveData class
