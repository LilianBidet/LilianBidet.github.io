reAuthAndDeleteWithEmail

private fun reAuthAndDeleteWithEmail(): Unit

Re-authenticates an "email/password" user in order to delete the account from the Firebase Authentication system.

This method creates an AlertDialog and shows it on the screen to ask the user for his or her credentials. Then, it retrieves the AuthCredential using the appropriate fields with the EmailAuthProvider.getCredential method. When it's donne :

  • First it removes user data from the Firebase Database with the DocumentReference.delete method.
  • Then, it removes user data from the Firebase Storage with the StorageReference.delete method.
  • Finally, it re-authenticates the user with the reauthenticate method and delete the account from the Firebase Authentication system with the delete method
