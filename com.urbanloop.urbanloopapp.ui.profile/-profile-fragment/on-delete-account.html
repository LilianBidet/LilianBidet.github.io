onDeleteAccount

private fun onDeleteAccount(): Unit

Starts the account's deletion flow.

This method creates an AlertDialog and shows it on the screen to warn the user about what the deletion involves. Then :

  • If the user clicks on the "OK" button, it calls deleteAccount
  • If the user clicks on the "Cancel" button, it cancels the dialog.
