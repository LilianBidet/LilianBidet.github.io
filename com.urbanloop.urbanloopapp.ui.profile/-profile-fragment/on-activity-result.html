onActivityResult

fun onActivityResult(requestCode: Int, resultCode: Int, data: Intent?): Unit

onActivityResult overridden to get the result from the google sign in Intent

This method gets the GoogleSignInAccount returned from launching the Google sign in Intent from GoogleSignInClient.getSignInIntent Then, it starts getGoogleAuthCredentialAndDeleteAccount with it in order to delete the account from the application.

See Authenticate Using Google Sign-In on Android for more information

Parameters

requestCode - The integer request code originally supplied to startActivityForResult(), allowing you to identify who this result came from.

resultCode - The integer result code returned by the child activity through its setResult().

data - An Intent, which can return result data to the caller (various data can be attached to Intent "extras").
