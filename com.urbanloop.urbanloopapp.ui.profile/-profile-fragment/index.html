ProfileFragment

class ProfileFragment : Fragment

A Fragment class providing a user interface to see and manage the user's profile on the application.

This fragment provides a user interface to reach Urbanloop stations from a map view. The interface is made up of the following elements :

  • A section where the user can see personal information
  • A section where the user can see and manage the balance of the account
  • A section where the user can manage some settings about the account : the default capsule type, the personal information (which goes to the PersonalInformationFragment) and the payments ways (which does nothing for now)

To match the recommended app architecture, this class interacts with the ProfileViewModel to handle it's data logic.
