updateFirebasePersonalInfo

fun updateFirebasePersonalInfo(user: FirebaseUser, firstName: String, lastName: String, displayName: String, email: String): Unit

Updates the data of the provided user in the Firebase database with the provided strings.

This method calls FirebaseRepository.getUserDataReferenceFromUser with the provided FirebaseUser and updates the corresponding document in the Firebase database with the update method.

Parameters

user - The FirebaseUser of whom it updates the data

firstName - The new first name to save in the database

lastName - The new last name to save in the database

displayName - The new display name to save in the database

email - The new email to save in the database
