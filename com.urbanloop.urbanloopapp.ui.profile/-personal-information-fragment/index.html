PersonalInformationFragment

class PersonalInformationFragment : Fragment

A Fragment class providing a user interface to see and manage user's personal information on the application.

This fragment provides a user interface to see and manage user's personal information on the application. The interface is made up of the following elements :

  • A section when the user can change personal information fields
  • A button allowing the user to save data entered in the appropriate fields

To match the recommended app architecture, this class interacts with the PersonalInformationViewModel to handle it's data logic.
