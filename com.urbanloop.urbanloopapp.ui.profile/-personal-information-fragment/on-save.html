private fun onSave(): Unit

Listen to the Save button

This method calls setOnClickListener from the button view to invoke a callback when the button is clicked. The callbacks calls PersonalInformationViewModel.updateFirebasePersonalInfo to update user information in the Firebase database with the current values in the appropriate fields.
