checkIfUserIsAuthenticated

private fun checkIfUserIsAuthenticated(): Unit

Checks if a user is authenticated and react accordingly.

This method calls SplashViewModel.checkIfUserIsAuthenticated and observe SplashViewModel.isUserAuthenticatedLiveData with an Observer. Then :

  • If the SplashViewModel.isUserAuthenticatedLiveData value tells that the user is not authenticated it calls goToAuthInActivity and finishes the activity
  • If the SplashViewModel.isUserAuthenticatedLiveData value tells that the user is authenticated, it calls getUserFromDatabase
