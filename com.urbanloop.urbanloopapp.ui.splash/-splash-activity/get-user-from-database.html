getUserFromDatabase

private fun getUserFromDatabase(uid: String): Unit

Gets user data from the database with the SplashViewModel

This method calls SplashViewModel.setUid to get user data from the Firebase database and observe SplashViewModel.userLiveData with an Observer. Then, when SplashViewModel.userLiveData is set, it calls goToMainActivity with it and finishes the activity

Parameters

uid - The user id of the user
