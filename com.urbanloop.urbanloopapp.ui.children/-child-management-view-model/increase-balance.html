increaseBalance

fun increaseBalance(userID: String): Unit

Increase the balance of a user corresponding to the provided user ID in the Firebase database

This method calls FirebaseRepository.getUserDataReferenceFromID and updates the corresponding document in the Firebase database with the update method.

The balance is increased by 10 here.

Parameters

userID - The uid of the FirebaseUser of whom it increases the balance.
