ChildManagementFragment()

A Fragment class providing a user interface to manage a child account.

This fragment provides a user interface to manage a child account made up of the following elements :

  • A section to show the child information
  • A section to manage the child balance
  • A section to manage which station is allowed for the child account

To match the recommended app architecture, this class interacts with the ChildManagementViewModel to handle it's data logic.
